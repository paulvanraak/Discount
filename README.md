# Bespaar met Donnie

Vergelijker voor financiële diensten en abonnementen in Nederland.

## Stack
- React 18 + Vite
- Tailwind CSS
- React Router v6
- Material Symbols Rounded

## Development
npm install && npm run dev

## Folder structure
- src/components/ — UI componenten
- src/pages/ — route-level pages
- src/data/ — categorieën en aanbieders
- src/services/ — affiliate, analytics, savings
- mobile/ — oude React Native code (niet aanraken)
